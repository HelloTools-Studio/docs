# 成为 HelloTools Cloud 的赞助者 {#become-a-sponsor}

目前我们的赞助方式有 3 种：爱发电发电、虚拟货币转账和微信收款

## 爱发电发电 {#afdian}

爱发电发电是针对本团队的赞助，发电后，您可以得到如下权益：

1. 申请加入团队（一起摸鱼）；

2. 在我们的[团队页面](/about/member)上留下您的名字。

如果您对此赞助位感兴趣，请单击[这个链接](https://afdian.net/a/xiaozhu2021)开始发电。

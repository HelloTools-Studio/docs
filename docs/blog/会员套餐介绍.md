## 套餐介绍

| 套餐名       | 普通用户 | 专业会员  | 超级会员 |
| ------------ | -------- | --------- | -------- |
| 去除广告     | 否       | 是        | 是       |
| 网页类型     | 静态\*   | 静态/托管 | 动态\*   |
| 总月度流量   | 20GB     | 50GB      | 80GB     |

| 普通用户 | 专业会员      | 超级会员      |
| -------- | ------------- | ------------- |
| 免费     | 49 元\/人\/年 | 88 元\/人\/年 |

## 会员常见问题

- Q：可以按月份/季度购买语雀会员吗？
- A：目前不支持按月份/季度购买语雀会员，只能年付购买。

- Q：流量如果超限怎么办？
- A：流量超限影响对应的服务的访问，您可以升级更高等级的会员享受更大的流量。
